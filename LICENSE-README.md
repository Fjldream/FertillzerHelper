# 激活码系统使用说明

## 用户激活流程

1. 用户打开应用，看到激活界面
2. 界面显示机器码（格式：XXXX-XXXX-XXXX-XXXX）
3. 用户点击"复制"按钮，通过微信/邮件发给你
4. 你使用 `license-generator.html` 生成激活码
5. 把激活码发给用户
6. 用户在应用中输入激活码，点击"激活"
7. 激活成功后，应用正常使用

## 管理员生成激活码

1. 用浏览器打开 `license-generator.html`
2. 填写：
   - **用户机器码**：用户发给你的机器码
   - **到期日期**：默认1年后，可自定义
   - **私钥**：复制 `../license-keys/private_pkcs8.pem` 的全部内容粘贴进去
3. 点击"生成激活码"
4. 复制生成的激活码发给用户

## 重要文件

- `../license-keys/private_pkcs8.pem` — **私钥（绝对保密，不要泄露）**
- `../license-keys/public.pem` — 公钥（已内置在应用中）
- `license-generator.html` — 激活码生成工具（仅你使用）

## 安全说明

- 私钥只有你持有，用户无法伪造激活码
- 激活码绑定机器码，无法在其他电脑使用
- 激活码包含到期时间，过期自动失效
- 激活信息存储在用户本地（localStorage）

## 常见问题

**Q: 用户换电脑了怎么办？**
A: 需要重新获取新电脑的机器码，生成新的激活码

**Q: 激活码可以给多人用吗？**
A: 不行，激活码绑定了机器码，只能在对应的电脑上使用

**Q: 如何延长用户的使用期限？**
A: 用相同的机器码，设置新的到期日期，重新生成激活码发给用户

**Q: 私钥丢了怎么办？**
A: 需要重新生成密钥对，所有用户需要重新激活
