# 微量元素功能更新说明 v2.0

## 🎉 新增功能

### 1. 新增原料时添加微量元素

**位置：** 原料管理 → 新增原料区域

**功能：**
- 点击"⚗️ 设置微量元素"按钮打开微量元素设置弹窗
- 可以设置常用微量元素（Ca、Mg、S、Fe、Mn、Zn、Cu、B、Mo、Co）
- **支持自定义添加任意微量元素**
- 设置完成后自动执行新增原料操作
- 也可以点击"快速新增（无微量元素）"直接添加原料

**使用流程：**
1. 填写原料基本信息（名称、价格、N、P、K、Cl）
2. 点击"⚗️ 设置微量元素"按钮
3. 在弹窗中设置常用微量元素含量
4. 如需添加自定义元素：
   - 输入元素符号（如：Si）
   - 输入元素名称（可选，如：硅）
   - 输入含量百分比
   - 点击"+ 添加"
5. 点击"确认"完成新增

### 2. 自定义微量元素

**功能：**
- 不再局限于预设的 10 种常用元素
- 用户可以自由添加任何微量元素
- 支持设置元素符号和中文名称
- 自定义元素会在编辑弹窗中单独显示

**应用场景：**
- 添加特殊元素：Si（硅）、Na（钠）、Cl（氯）等
- 添加有机物质：腐殖酸、氨基酸等
- 添加功能性成分：生长调节剂、增效剂等

### 3. 编辑弹窗增强

**位置：** 原料管理 → 原料表格 → 微量元素列 → 点击按钮

**新增功能：**
- 常用微量元素区域（预设 10 种）
- 自定义微量元素区域（显示已添加的自定义元素）
- 添加自定义元素功能（输入符号、名称、含量）
- 自动保存所有类型的微量元素

### 4. 详细报告支持自定义元素

**功能：**
- 详细报告弹窗自动识别所有微量元素（包括自定义的）
- 自动计算每种元素的总含量和质量
- 自定义元素显示符号或名称
- 按元素符号排序显示

  n: 0, p: 0, k: 0, cl: 0,
  trace: {
    Ca: 25,      // 常用元素

    '腐殖酸': 5  // 自定义元素（中文）

## 🔧 技术实现

### 核心函数更新

**trace.js 新增：**
- `openAddRawTraceEditor()` - 打开新增原料的微量元素设置弹窗
- `addCustomElementToNewRaw()` - 添加自定义元素到新增原料
- `removeCustomElement(index)` - 删除自定义元素
- `confirmAddRawTrace()` - 确认微量元素设置并执行新增
- `addCustomTraceElement(rawIndex)` - 在编辑弹窗中添加自定义元素

**trace.js 更新：**
- `openTraceEditor()` - 支持显示和编辑自定义元素
- `saveTraceElements()` - 保存所有类型的微量元素
- 详细报告函数 - 自动收集所有微量元素符号

**raw.js 更新：**
- `addRaw()` - 支持从 `window.pendingRawTrace` 获取微量元素数据

## 🚀 使用示例

### 示例 1：添加硅钙肥
```
1. 填写基本信息：
   - 名称：硅钙肥
   - 价格：3500
   - N: 0, P: 0, K: 0, Cl: 0

2. 点击"⚗️ 设置微量元素"

3. 设置常用元素：
   - Ca: 25%
   - Mg: 3%

4. 添加自定义元素：
   - 符号：Si
   - 名称：硅
   - 含量：15%
   - 点击"+ 添加"

5. 点击"确认"完成
```

### 示例 2：添加有机肥
```
1. 基本信息：有机肥料

2. 设置微量元素：
   - 常用：Fe: 0.5%, Mn: 0.3%

3. 自定义元素：
   - 符号：HA（或中文：腐殖酸）
   - 名称：腐殖酸
   - 含量：5%

   - 符号：AA（或中文：氨基酸）
   - 名称：氨基酸
   - 含量：3%
```

## 🎯 优势

1. **灵活性**：不受预设元素限制，可添加任意元素
2. **专业性**：支持特殊肥料配方（硅肥、有机肥等）
3. **易用性**：常用元素快速选择，自定义元素简单添加
4. **完整性**：详细报告自动显示所有元素

## 📝 注意事项

1. 元素符号建议使用标准化学符号（如 Si、Ca）
2. 也可以使用中文名称作为符号（如"腐殖酸"）
3. 自定义元素会在编辑弹窗中单独显示
4. 详细报告会自动识别并显示所有元素
5. 建议为自定义元素设置名称，便于识别
